# AI Planner

Personal AI assistant for tasks, calendar, habits and notes.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Create env file
cp .env.example .env.local
# Add your OPENAI_API_KEY to .env.local

# 3. Run locally
npm run dev
# Open http://localhost:3000
```

## Deploy to Vercel

1. Push to GitHub
2. Connect repo on vercel.com
3. Add OPENAI_API_KEY in Environment Variables
4. Deploy → get your live URL
